Run the test suite for the current project.

## Instructions

1. Identify the testing framework used in the project (pytest, unittest, jest, mocha, etc.)
2. Check for test configuration files (pytest.ini, jest.config.js, etc.)
3. Run the appropriate test command based on the framework
4. Report test results including:
   - Total tests run
   - Passed/failed counts
   - Any failing test details
   - Test coverage percentage if available

## Common Test Commands

- Python (pytest): `pytest` or `python -m pytest`
- Python (unittest): `python -m unittest discover`
- JavaScript (jest): `npm test` or `yarn test`
- JavaScript (mocha): `npm test`
- Go: `go test ./...`
- Rust: `cargo test`

## Output

Provide a summary of:
- Test framework identified
- Command executed
- Results summary
- Any failures that need attention
